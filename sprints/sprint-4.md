# Sprint 4: Frontend 분석 UI 및 통합

## 목표

분석 결과 UI 및 전체 플로우 통합

---

## 작업 목록

### Frontend

- [ ] 글 제출 버튼 및 확인 모달
- [ ] 분석 진행 중 UI (로딩 상태, Polling)
- [ ] 분석 결과 페이지
  - [ ] AI 탐지 점수 시각화 (게이지, 퍼센트)
  - [ ] 품질 피드백 표시
- [ ] Rate Limit 초과 시 안내 UI
- [ ] 분석 실패 시 에러 UI
- [ ] 제출 이력 표시 (글 목록에서 상태 확인)

### 통합 테스트

- [ ] 전체 플로우 E2E 테스트 (회원가입 → 글 작성 → 제출 → 결과 확인)
- [ ] 에러 케이스 테스트

### 문서화

- [ ] API 문서 정리 (OpenAPI/Swagger)
- [ ] 로컬 개발 환경 README 작성
